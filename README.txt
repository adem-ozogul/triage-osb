1. Komut satırında bu dosyaları aynı klasöre çıkarın.
2. Sanal ortam kurun ve bağımlılıkları yükleyin:
   python3 -m venv .venv
   source .venv/bin/activate   (Windows için: .venv\Scripts\Activate.ps1)
   pip install -r requirements.txt
3. Uygulamayı başlatın:
   streamlit run app.py
4. Tarayıcıda açılan sayfadan videoyu yükleyip anketi doldurun.
